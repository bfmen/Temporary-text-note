# PHP 临时文本便签

一个轻量级的临时文本便签系统，所有笔记保存为文本文件并支持密码保护功能。

## 📌 核心特性

- 📝 纯文本存储 - 无需数据库
- 🔐 基础密码保护（通过文件头标记实现）
- ⚖️ 轻量化设计（压缩后约10KB）
- 🛠️ 极简版本近期发布

> ⚠️ 注意：密码功能仅限制访问，**不会加密内容，新版本会加密内容**

## 🖥️ 服务器要求

- Nginx（需启用 ngx_http_rewrite_module + PHP）

- 可写的 `_notes` 目录

## 🚀 快速安装

1. **部署代码**  
   将项目文件复制到网站根目录

2. **配置重写规则**  
   **Nginx 配置**：
   ```nginx
   location / {
       rewrite ^/([a-zA-Z0-9_-]+)$ /index.php?note=$1;
   }
